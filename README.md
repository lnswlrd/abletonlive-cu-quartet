# CU-Quartet - Ableton Extension

<img width="770" height="418" alt="cu-quartet-screenshot" src="https://github.com/user-attachments/assets/dea67b37-fc57-42c9-b56f-237342d54923" />


CU-Quartet is an extension for Ableton Live, built using the Ableton Extensions SDK. It rewrites midi clips with quartet harmony and revoicing for up to four voices/parts. 

## Requirements

* **Ableton Live 12 Suite** (Version 12.4.5 beta or later)
* **Extensions enabled** in Live

## Installation

1. Go to the **Releases** section of this repository and download the latest `cu-quartet.ablx` file.
2. Open Ableton Live and navigate to **Settings** → **Extensions**.
3. Install/add the downloaded `.ablx` file.
4. To use it, right-click a MIDI clip in your Set and select **CU-Quartet** from the context menu to generate your parts.
